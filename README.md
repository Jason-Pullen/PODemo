# PO Demo #

## Description ##
A small demo on exchanging different currencies

## Instructions ##
 - Clone repo
 - Install deps and run server
 
 ```
 npm install && cd server  && npm install && cd .. && node ./server/server.js
 
 ```

 - Run on simulator ios 
 
 ``` 
 npm run ios
 ```

  
or visit [Web Version](http://localhost:8081)
